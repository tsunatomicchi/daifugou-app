# つなサポ大富豪（試作版）

GitHub Pages でそのまま動く HTML / CSS / JavaScript の大富豪アプリです。

## 今回追加した内容

- 場札より弱いカードを出せない基本ルールを追加
- 場と同じ枚数・同ランク判定の基本ルールを追加
- スート表示を改善（左上・中央・右下に数字とスート表示）
- CPUのカード提出表示を追加（誰が何を出したか、パス履歴）
- CPUの残り手札枚数表示を追加
- 簡易CPU対戦を追加（出せる中で一番弱いカードを出す）
- 縛りルールを追加（同スート連続で縛り発生、場流しで解除）
- 勝利時「あなたの上がりです！」と順位確定表示を追加
- 順位確定後のカード交換フェーズを追加

## 現在の仕様

- 4画面構成（トップ / ログイン / ルーム一覧 / ゲーム）を維持
- 8切り・革命（革命返し）に対応
- 現在は GitHub Pages 上で動く試作版
- オンライン対戦と Google ログインは今後実装予定

## 起動方法

`index.html` を直接ブラウザで開くか、以下でローカルサーバー起動:

```bash
python3 -m http.server 8080
```
