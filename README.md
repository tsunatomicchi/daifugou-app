# つなサポ大富豪（1人用・動作確認版）

つなサポサイトから使える大富豪アプリの**1人用動作確認版**です。  
GitHub Pages でそのまま動く HTML / CSS / JavaScript で構成しています。

## 今回追加した内容

- トランプ札の見た目を改善（白カード、角丸、薄い影、スート色分け、左上/右下の数字・スート、中央スート、選択時の浮き表示）
- 8切りに対応（8を含むカードを出すと「8切り！」表示後に場を流す）
- 革命・革命返しに対応（同数4枚で革命、再度4枚で元に戻る）
- 現在の状態（通常 / 革命中）をゲーム画面に表示

## 現在の仕様

- 1人用の動作確認版です
- オンライン対戦は今後実装予定です
- 4画面構成（トップ / ログイン / ルーム一覧 / ゲーム）とナビゲーションは維持しています

## 起動方法

### 1) ファイルをそのまま開く

`index.html` をブラウザで直接開いて動作確認できます。

### 2) ローカルサーバー（推奨）

```bash
python3 -m http.server 8080
```

- http://localhost:8080
