# つなサポ大富豪（試作版）

GitHub Pages でそのまま動く HTML / CSS / JavaScript の大富豪アプリです。

## 対応内容

- プレイヤー1人（「あなた」）＋CPU3人（CPU1/CPU2/CPU3）の4人対戦に対応
- プレイヤーは自分の手札を選んで「カードを出す / パス」を操作可能
- CPUは自動で行動し、出せる中で一番弱いカードを出す（なければパス）
- 場札より弱いカードは出せない
- 8切り、革命、革命返し、縛り、パス、場流しに対応
- 勝利時表示（あなたの上がり表示）と順位確定表示に対応
- 順位確定後のカード交換フェーズ（1位↔4位で2枚、2位↔3位で1枚）に対応
- 4画面構成（トップ / ログイン / ルーム一覧 / ゲーム）を維持
- すべてのカードでスートを表示（左上・中央・右下）

## 現在の仕様

- 現在は GitHub Pages 上で動く試作版です
- オンライン対戦と Google ログインは今後実装予定です

## 起動方法

`index.html` を直接ブラウザで開くか、以下でローカルサーバー起動:

```bash
python3 -m http.server 8080
```
