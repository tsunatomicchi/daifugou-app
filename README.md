# つなサポ大富豪（試作モック）

つなサポサイトから使えるオンライン大富豪アプリの**見た目・画面遷移確認用モック**です。

## 実装内容

- 通常の HTML / CSS / JavaScript で作成
- 小中学生向けの明るいUI
- 以下4画面をモックとして実装
  - トップ画面
  - ログイン画面
  - ルーム一覧
  - ゲーム画面
- 画面下部ナビゲーションとボタンで画面遷移

## 起動方法

### 1) ファイルをそのまま開く方法（最速）

`index.html` をブラウザで直接開くと動作確認できます。

### 2) ローカルサーバーで起動（推奨）

```bash
python3 -m http.server 8080
```

ブラウザで以下にアクセス:

- http://localhost:8080

## 将来のGoogle認証・許可アカウント制限に向けた構成

現在はモックですが、将来を見据えて以下の方針で拡張しやすい構成にしています。

1. `app.js` のルーティング/状態管理を認証状態と連動可能な形にしている
2. ログイン画面に `Googleでログイン（予定）` ボタンを配置
3. 今後、以下を追加予定
   - Google OAuth 2.0 (Google Identity Services)
   - バックエンドでIDトークン検証
   - 許可メールアドレス/ドメインのホワイトリスト制御
   - ルーム入室時の権限チェック

## 今後必要な設定（本番化時）

- Google Cloud プロジェクト作成
