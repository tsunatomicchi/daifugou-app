# つなサポ大富豪（試作版）

GitHub Pages でそのまま動く HTML / CSS / JavaScript の大富豪アプリです。

## 対応内容

- プレイヤー1人（「あなた」）＋CPU3人（CPU1/CPU2/CPU3）の4人対戦に対応
- プレイヤーは自分の手札を選んで「カードを出す / パス」を操作可能
- CPUは自動で行動し、単カード・同数複数枚・階段の候補から出せる中で一番弱い手を出す（なければPASS）
- 全員がパスしたときに自動で場を流し、最後に出した人から再開
- CPUの行動をカード表示で見える化（PASS表示・履歴・残り枚数表示）
- 場札より弱いカードは出せない
- 8切り、革命、革命返し、通常縛り、階段縛り、パス、場流しに対応
- カジノのポーカー台風の場エリアを追加
- 勝利時表示（あなたの上がり表示）と順位確定表示に対応
- 順位確定後のカード交換フェーズを維持
- 効果音（play/error/turn/win）をWeb Audio APIで追加
- 効果音ON/OFF切り替えに対応
- 4画面構成（トップ / ログイン / ルーム一覧 / ゲーム）を維持
- すべてのカードでスートを表示（左上・中央・右下）

## 現在の仕様

- 現在は GitHub Pages 上で動く試作版です
- オンライン対戦と Google ログインは今後実装予定です

## 起動方法

`index.html` を直接ブラウザで開くか、以下でローカルサーバー起動:

```bash
python3 -m http.server 8080
```
