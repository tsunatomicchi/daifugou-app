# つなサポ大富豪（1人用動作確認版）

つなサポサイトから使える大富豪アプリの**1人用・動作確認版**です。  
現在は GitHub Pages で動作する HTML / CSS / JavaScript のみで構成しています。

## 現在できること

- 手札カードをクリックして選択 / 選択解除
- 選択カードを「カードを出す」で場に移動
- 未選択で「カードを出す」を押すと「カードを選んでください」を表示
- 「パス」で「パスしました」を表示
- 「新しく配る」で手札と場札をリセット
- 画面下ナビゲーションを小画面でも崩れにくいレイアウトに調整

## 起動方法

### 1) ファイルをそのまま開く方法（最速）

`index.html` をブラウザで直接開くと動作確認できます。

### 2) ローカルサーバーで起動（推奨）

```bash
python3 -m http.server 8080
```

ブラウザで以下にアクセス:

- http://localhost:8080
